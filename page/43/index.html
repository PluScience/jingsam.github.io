Cygwin 安装 PostGIS

2016-01-17

PostGIS 是一款基于 PostgreSQL 的插件，提供对空间数据的操作。虽然 PostgreSQL 本身可以存储空间数据，但 PostGIS 扩展可以使用户直接使用 SQL 语句进行空间查询、分析和处理。

Cygwin 的官方源中并没有 PostGIS 的二进制发行包，所以要在 Cygwin 上使用 PostGIS 必须自行编译。
